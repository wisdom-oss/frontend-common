src/lib/http-context/send-auth.ts

SEND_AUTH
Default value : new HttpContextToken<undefined | boolean>(() => undefined)

HttpContextToken used to determine if the authorization header should be sent.

As long as this context is set to undefined it is implied by USE_API_URL to be true.

Defaults to undefined.

src/lib/http-context/use-api-url.ts

USE_API_URL
Type : miscellaneous
Default value : new HttpContextToken<boolean>(() => false)

HttpContextToken used to determine if the api url should be used.

Implies SEND_AUTH. Setting SEND_AUTH to false will deny filling the Authorization header.

Defaults to false.

src/lib/http-context/use-error-curtain.ts

USE_ERROR_CURTAIN
Default value : new HttpContextToken<string | boolean>(() => true)

HttpContextToken used to determine if requests that error should show the error message on curtain covering the whole main container.

Disable this if you want to handle errors yourself.

Defaults to true.

src/lib/http-context/use-loader.ts

USE_LOADER
Default value : new HttpContextToken<string | boolean>(() => false)

HttpContextToken used to determine if loader should be shown. A given string will be run through the translator and therefore may get translated.

Useful for longer loading times.

Defaults to false.
